# TEKsystems Java Developer - Case Study
This is an application about bears.

## Technologies Used

* Spring MVC
* Spring Data JPA with Hibernate
* Spring Security
* Java Bean Validation and Hibernate Validator
* JUnit
* Thymeleaf
* MariaDB
* H2 for testing
* Bootstrap
* HTML
* CSS
